# DigitalSolutions - Fullstack Project & Admin Panel

A complete full-stack web application with a customer-facing landing page and an admin dashboard for managing content.

## ‚ú® Features

### **Landing Page**
- **Our Projects Section**: Displays projects with images, names, descriptions, and locations
- **Happy Clients Section**: Shows client testimonials with photos and designations
- **Contact Form**: Functional form with validation for collecting inquiries
- **Newsletter Subscription**: Email subscription system

### **Admin Panel**
- **Project Management**: Add/delete projects with image uploads
- **Client Management**: Add/delete client testimonials with photos
- **Contact Management**: View all contact form submissions
- **Subscriber Management**: View all newsletter subscribers

### **Backend API**
- RESTful API built with Node.js & Express
- MongoDB database with Mongoose ODM
- Image upload & processing with Sharp.js
- CORS-enabled for cross-origin requests
